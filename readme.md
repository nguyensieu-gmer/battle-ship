## About this project

- This project is about classic game: Battleship
- The tools I use are all listed in package.json
- the game has two parts: placing ships and shoting
- It is easy to play but hard to create, at least it is hard for me

## User instructions

- you must fork it into your local repository, and run npm i to download all pakages
- Run npm run dev to start on your localhost

- First you need to place the ship in grid that you think are hard to reach
- You can put a ship by choose the orientation in two top button
  , drag and drop the ship on the grid
- You can reset in reset button on bottom right if you want
- Once you are satisfied, click confirm button to play the game
